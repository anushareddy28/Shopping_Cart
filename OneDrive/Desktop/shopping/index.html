About This System

SmartCart is a JSON API based e-commerce system built with PHP, MySQL, and vanilla JavaScript. It demonstrates REST API communication, session-based authentication, server-side cart management, coupon validation, transactional order placement, inventory tracking, and a complete admin dashboard. Designed as an academic project showcasing full-stack web development principles.
